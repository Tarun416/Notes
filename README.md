# NoteApp

Functionality
 
* Create Notes with title and description
* Update 
* Delete

Tech Used : 
* Kotlin
* Data persist locally ( Room )
* Completely offline
* Unit test
* Clean architecture
* Dagger
* MVVM
* LiveData

Ktlint and lint used to assure proper coding style.
